RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Methods
Index /blogs GET Dispay All Dogs Blog.find()
New /blogs/new GET Show new blog form N/A
Create /blogs POST Create a new dog, then redirect somewhere Blog.create()
Show /blogs/:id GET Show info about one specific blog Blog.findById()
Edit /blogs/:id/edit GET Show edit form for one blog Blog.findById()
Update /blogs/:id PUT Update a particular dog, then redirect somehwere Blog.findByIdandUpdate()
Destroy /blogs/:id DELETE Delete a particular blog, then redirect somewhere Blog.findByIdandDelete()
